Télétype pour Atom

De grandes choses se produisent lorsque les développeurs travaillent ensemble - de l'enseignement et du partage des connaissances à la création de meilleurs logiciels. Teletype pour Atom rend la collaboration sur du code aussi simple que de coder seul, directement depuis votre éditeur.

GitHub pour Atom

Un éditeur de texte est au cœur de la boîte à outils d'un développeur, mais il ne fonctionne généralement pas seul. Travaillez avec Git et GitHub directement depuis Atom avec le package GitHub.


Créez de nouvelles branches, organisez et validez, envoyez et tirez, résolvez les conflits de fusion, affichez les demandes d'extraction et bien plus encore, le tout depuis votre éditeur. Le package GitHub est déjà fourni avec Atom, vous êtes donc prêt à partir!
